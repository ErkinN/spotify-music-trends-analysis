# Spotify Music Trends Analysis (2009–2025)

## Overview
This project explores how music characteristics and popularity on Spotify have evolved over time using descriptive analytics. The analysis focuses on track, artist, and genre trends without applying machine learning techniques.

## Tools
- Python (pandas, matplotlib)
- Jupyter Notebook

## Key Questions
- How has track popularity changed over time?
- Are modern tracks different from classic tracks?
- Do prolific artists outperform others in popularity?
- Which genres dominate Spotify, and which perform best?

## Key Insights
- Track duration and explicit content proportions remain stable across eras.
- Year-by-year analysis reveals nuances hidden in aggregate comparisons.
- Artist popularity is not strongly tied to catalog size.
- A small number of genres dominate the Spotify catalog.

## Data
Source: Kaggle – Spotify Global Music Dataset (2009–2025)

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the Jupyter notebook
